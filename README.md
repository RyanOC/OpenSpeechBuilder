# Open Speech Builder

A web-based Augmentative and Alternative Communication (AAC) soundboard designed to help people communicate using voice synthesis. Features customizable buttons, sentence building, and custom image creation.

## Who is this for?

- **Non-speaking individuals** who need alternative ways to communicate
- **People with speech difficulties** due to conditions like autism, cerebral palsy, or stroke
- **Language learners** who want to practice pronunciation
- **Caregivers and therapists** working with individuals who use AAC
- **Anyone** who needs quick access to common phrases

## Technical Features

- **Works offline** - Uses browser's built-in speech synthesis
- **No installation** - Runs directly in web browsers
- **Responsive design** - Works on desktop, tablet, and mobile
- **Accessibility focused** - Keyboard navigation and screen reader support
- **Theme support** - Dark and light modes
- **Local storage** - Configurations saved in browser

## Development
